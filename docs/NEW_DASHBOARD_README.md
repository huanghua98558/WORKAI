# 新仪表盘访问指南

## 访问方式

### 开发环境
访问：http://localhost:5000/new-dashboard

### 生产环境
根据您的部署配置访问相应地址。

## 主要功能

### 📊 系统监控
- 实时系统状态
- 运行时间统计
- 在线机器人数量
- 今日回调统计

### 🤖 机器人管理
- 查看所有机器人状态
- 监控机器人健康度
- 查看消息处理量
- 成功率统计

### 🚨 告警分析
- 告警概览统计
- 告警级别分布
- 影响范围分析
- 关键指标追踪

### 📈 活跃排行
- Top活跃群组
- Top活跃用户
- 活跃度分级
- 消息量统计

### 💬 实时会话
- 查看活跃会话
- 会话状态标识
- 消息统计
- 最后更新时间

## 操作说明

### 刷新数据
点击右上角的"刷新"按钮，立即更新所有数据。

### 查看详情
每个模块显示关键指标，点击卡片可以查看更多详细信息（开发中）。

### 自动刷新
仪表盘每60秒自动刷新一次数据，保持信息实时性。

## API接口

新仪表盘已对接以下API接口：

### 监控接口
- `GET /api/monitoring/summary` - 监控摘要
- `GET /api/monitoring/robots-status` - 机器人状态
- `GET /api/monitoring/active-groups` - 活跃群组
- `GET /api/monitoring/active-users` - 活跃用户

### 告警接口
- `GET /api/alerts/analytics/overview` - 告警概览
- `GET /api/proxy/admin/sessions/active` - 活跃会话
