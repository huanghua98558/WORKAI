# 机器人管理中心 - 整合版

## 🎯 功能说明

所有机器人相关的功能已整合到一个页面中，使用内部标签页切换不同模块。

## 📋 功能模块

### 1. 机器人列表
- 查看所有机器人列表
- 显示机器人状态（在线/离线）
- 支持刷新列表
- 暂时为简化版本

### 2. 分组管理
- 创建和管理机器人分组
- 配置分组图标、颜色、优先级
- 支持多种路由策略
- 负载均衡配置

### 3. 角色管理
- 创建和管理机器人角色
- 细粒度权限控制
- 配置操作范围
- 速率限制设置

### 4. 指令发送
- 向在线机器人发送指令
- 支持 8 种指令类型
- 实时查看执行状态
- 失败自动重试

### 5. 监控大屏
- 实时监控所有机器人状态
- 健康度、成功率、负载可视化
- 自动刷新（10秒）
- 错误统计和分析

## 🚀 使用方法

1. 打开主页面：http://localhost:5000
2. 点击"机器人管理"标签
3. 在内部标签页中切换不同功能模块
