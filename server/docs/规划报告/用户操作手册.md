## 1. 系统简介

### 1.1 系统概述

WorkTool AI 中枢系统是一个基于企业微信的智能客服系统，通过AI机器人自动处理客户咨询、问题解答、售后支持等业务，同时支持人工介入、协同分析、告警管理等功能。

### 1.2 核心功能

```
智能客服
- 自动回复客户咨询
- 意图识别
- 情感分析
- 多轮对话

告警管理
- 用户投诉告警
- 售后任务告警
- 低满意度预警
- 人工介入请求

售后管理
- 售后任务创建
- 任务分配
- 任务跟踪
- 腾讯文档同步

协同分析
- 工作人员监控
- 用户满意度分析
- 协同决策
- 质量评估
```

### 1.3 适用场景

```
企业微信客服群
- 产品咨询
- 问题解答
- 售后服务
- 投诉处理

售后服务
- 退货退款
- 产品问题
- 服务问题
- 订单问题

协同工作
- 多人协作
- 任务分配
- 进度跟踪
- 质量监控
```

## 2. 快速开始

### 2.1 系统登录

#### 步骤1：访问系统

```
1. 打开浏览器
2. 输入系统地址：https://worktool.company.com
3. 点击"登录"按钮
```

#### 步骤2：输入账号信息

```
用户名: 输入您的用户名
密码: 输入您的密码

如果忘记密码，点击"忘记密码"进行重置
```

#### 步骤3：验证登录

```
1. 如果启用了双因素认证，输入验证码
2. 点击"登录"
3. 登录成功后进入系统首页
```

### 2.2 首次使用

#### 个人设置

```
1. 点击右上角头像
2. 选择"个人设置"
3. 完善个人信息：
   - 真实姓名
   - 联系电话
   - 邮箱地址
   - 头像上传
4. 点击"保存"
```

#### 修改密码

```
1. 进入"个人设置"
2. 点击"修改密码"
3. 输入：
   - 当前密码
   - 新密码
   - 确认新密码
4. 点击"提交"
```

#### 告警处理流程

```
1. 确认告警
   点击"确认"按钮，确认已接收告警
   系统记录确认时间和确认人

2. 处理告警
   点击"处理"按钮，开始处理告警
   可以添加处理备注

3. 解决告警
   点击"解决"按钮，标记告警已解决
   需要填写解决说明

4. 取消告警
   点击"取消"按钮，取消告警
   需要填写取消原因
```

## 4. 常见操作

### 4.1 查看会话

```
步骤：
1. 点击"会话管理"菜单
2. 在会话列表中找到目标会话
3. 点击"查看详情"
4. 查看会话的消息记录
5. 可以发送消息或创建任务
```

### 4.2 处理告警

```
步骤：
1. 点击"告警管理"菜单
2. 在告警列表中找到目标告警
3. 点击"确认"接收告警
4. 查看告警详情和关联会话
5. 点击"处理"开始处理
6. 处理完成后点击"解决"
7. 填写解决说明
```

### 4.3 分配任务

```
步骤：
1. 点击"任务管理"菜单
2. 在任务列表中找到目标任务
3. 点击"分配"
4. 选择处理人
5. 点击"确认"
6. 系统通知处理人
```

### 4.4 同步到腾讯文档

```
自动同步：
1. 创建任务时勾选"同步到腾讯文档"
2. 系统自动同步到指定文档

手动同步：
1. 点击"腾讯文档集成"菜单
2. 找到需要同步的任务
3. 点击"重新同步"
4. 等待同步完成
```

### 4.5 查看用户满意度

```
步骤：
1. 点击"协同分析"菜单
2. 选择"用户满意度"
3. 选择分析维度（用户/时间/群组）
4. 查看满意度数据和图表
5. 可以导出报告
```

## 5. 常见问题

### 5.1 登录相关

**Q：忘记密码怎么办？**

A：点击登录页面的"忘记密码"，输入注册邮箱，按照邮件指引重置密码。

**Q：账号被锁定怎么办？**

A：联系系统管理员解锁，或者等待30分钟后自动解锁。

**Q：登录后立即被登出？**

A：可能是网络问题或会话超时，请检查网络连接后重新登录。

### 5.2 功能使用

**Q：为什么看不到某些会话？**

A：请检查您的权限设置，只有有权限的会话才会显示。

**Q：如何批量处理任务？**

A：当前版本支持逐个处理任务，批量处理功能将在后续版本中提供。

**Q：腾讯文档同步失败怎么办？**

A：请检查腾讯文档权限配置，或点击"重新同步"重试。

### 5.3 数据导出

**Q：如何导出会话记录？**

A：在会话详情页面，点击"导出"按钮，选择导出格式。

**Q：如何导出统计数据？**

A：在统计页面，点击"导出报告"，选择导出范围和格式。

### 5.4 技术支持

**Q：遇到问题如何联系技术支持？**

A：
